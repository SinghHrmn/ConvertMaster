About Us

Convert Master is a free open source online software that converts XML, CSV, JSON to each other. Convert Master is easy to use and powerful enough to handle a large amount of data. One can used it for downloading the converted file and also to preview the converted file on the go. Convert Master is a project build under CESS Summer Of Code 2.0 by Team D'Coders. Its open source nature allows you to contribute and make it even more incredible. See Documentation for help!
